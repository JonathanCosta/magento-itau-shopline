Shopline Itau (Magento)
=====================

Modulo de Pagamento do Itau Shopline para o Magento

Como Usar
=====================

Basta Copiar o conteúdo para a pasta ROOT do seu magento e habilitar o módulo nas configurações de pagamentos.
